
  ```sh
  $ docker exec -it trybers_and_dragons bash
  ```

  ```sh

  ```
  
  - Access the newly created directory:
  ```sh
  $ cd Project-Trybers-and-dragons
  ```
  
  - Install dependencies:
  ```sh
  $ npm install
  ```
